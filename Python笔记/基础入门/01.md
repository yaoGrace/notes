## *注意事项*

注意： Mac 和 win 输入的是有些不同的

win输入的是：python 回车

mac输入的是：python3 回车

如果想让Mac的输入和win系统的一致，可以在终端中输入一条命令 修改

输入命令为：
```bash
echo 'alias python="python3"' >> .bash_profile 

输入后回车，关闭终端，然后重新打开终端输入命令：python 即可
```
