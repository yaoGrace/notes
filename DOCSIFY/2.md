# 二、定制导航栏及侧边栏

 * 2.1、 ***script*** 配置

```
  <script>
    window.$docsify = {
      name: '',
      repo: '',
      loadNavbar: true,
      loadSidebar: true,
      maxLevel: 2,
      subMaxLevel: 4,
      mergeNvabar: true
    }
  </script>
```

* 添加 ***_sidebar.md*** 文件来配置侧边栏

  ```
  * DOCSIFY学习笔记
    * [一、初始化项目](DOCSIFY/1.md)
    * [二、侧边栏配置](DOCSIFY/2.md)
  ```

